CUS1172 Bootstrap Profile Website

📌 Project Overview

This project is a responsive personal profile website created using HTML5, CSS3, and Bootstrap 5 for the CUS1172 Web Development course at St. John’s University.

The website showcases personal information, education background, skills, interests, and contact details while demonstrating modern web design principles and responsive layout techniques.

🚀 Features

Responsive layout using Bootstrap 5

Navigation bar with working tabs

Mobile, tablet, and desktop compatibility

Custom CSS styling

Organized page sections:

Home

Education

Skills

Experience

Interests

Contact

Clean and modern UI design

GitHub Pages deployment
